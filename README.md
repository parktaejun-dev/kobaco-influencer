## 🔧 설치 방법

### 1단계: Python 설치 확인
먼저 Python이 설치되어 있는지 확인하세요. (Python 3.8 이상 필요)

```bash
python --version
```

### 2단계: 프로젝트 폴더로 이동
파일들이 있는 폴더로 이동합니다.

```bash
cd 프로젝트폴더경로
```

### 3단계: 필요한 패키지 설치
아래 명령어를 실행하여 필요한 패키지를 설치합니다.

```bash
pip install -r requirements.txt
```

또는 개별적으로 설치:

```bash
pip install streamlit requests
```

## 🔑 API 키 발급

YouTube Data API를 사용하려면 Google Cloud Console에서 API 키를 발급받아야 합니다.

### 1단계: Google Cloud Console 접속
- https://console.cloud.google.com 방문
- Google 계정으로 로그인

### 2단계: 새 프로젝트 만들기
1. 상단의 프로젝트 선택 드롭다운 클릭
2. '새 프로젝트' 버튼 클릭
3. 프로젝트 이름 입력 (예: "youtube-influencer-search")
4. '만들기' 버튼 클릭
5. 프로젝트가 생성되면 해당 프로젝트를 선택

### 3단계: YouTube Data API v3 활성화
1. 왼쪽 메뉴에서 'API 및 서비스' → '라이브러리' 선택
2. 검색창에 "YouTube Data API v3" 입력
3. 'YouTube Data API v3' 클릭
4. '사용' 또는 '활성화' 버튼 클릭

### 4단계: API 키 만들기
1. 왼쪽 메뉴에서 'API 및 서비스' → '사용자 인증 정보' 선택
2. 상단의 '+ 사용자 인증 정보 만들기' 버튼 클릭
3. 'API 키' 선택
4. API 키가 생성되면 **복사**해두세요 (나중에 다시 확인 가능)

### 5단계: API 키 제한 설정 (선택사항, 보안 강화)
1. 생성된 API 키 옆의 편집(연필) 아이콘 클릭
2. 'API 제한사항'에서 'API 키 제한' 선택
3. 'YouTube Data API v3'만 선택
4. '저장' 클릭

**중요:** API 키는 무료 할당량이 있습니다. (하루 10,000 단위)
- 채널 정보 조회: 약 3-5 단위 사용
- 따라서 하루에 약 2,000~3,000개 채널 조회 가능

## ▶️ 실행 방법

터미널(명령 프롬프트)에서 아래 명령어를 입력합니다:

```bash
streamlit run youtube_influencer_app.py
```

실행하면 자동으로 브라우저가 열리고 웹 애플리케이션이 실행됩니다.

만약 자동으로 열리지 않으면, 터미널에 표시된 주소(보통 http://localhost:8501)를 브라우저에 입력하세요.

## 📖 사용 방법

### 1단계: API 키 입력
1. 왼쪽 사이드바에 있는 'YouTube API 키를 입력하세요' 입력창 찾기
2. 발급받은 API 키 붙여넣기
3. Enter 키 누르기

### 2단계: 유튜브 채널 URL 입력
지원되는 URL 형식:
- `https://www.youtube.com/@channelname` (추천)
- `https://www.youtube.com/channel/UC...` (채널 ID)
- `https://www.youtube.com/c/channelname`
- `https://www.youtube.com/user/username`

예시:
```
https://www.youtube.com/@netflix
https://www.youtube.com/channel/UCwB45wkYLX7vYI7wjrYX5Fg
```

### 3단계: 결과 확인
입력하면 자동으로 다음 정보가 표시됩니다:
- ✅ 채널 기본 정보 (이름, 썸네일, 설명)
- 📊 채널 통계 (구독자, 동영상 수, 총 조회수)
- 🎥 최근 영상 분석 (평균 조회수, 참여율)
- 💰 광고 비용 추정

## 💡 비용 산출 로직

### 기본 공식
```
최종 비용 = 기본 비용 × 구독자 보정 × 참여율 보정
```

### 1. 기본 비용
```
기본 비용 = 평균 조회수 × 10원
```

예시:
- 평균 조회수가 10만이면 → 100만원
- 평균 조회수가 50만이면 → 500만원

### 2. 구독자 수 보정
채널의 영향력을 반영합니다:

| 구독자 수 | 보정 배율 | 설명 |
|---------|---------|------|
| 10만 미만 | ×0.8 | 소형 채널 |
| 10만~50만 | ×1.0 | 중형 채널 |
| 50만~100만 | ×1.2 | 대형 채널 |
| 100만 이상 | ×1.5 | 메가 채널 |

### 3. 참여율 보정
시청자의 반응도를 반영합니다:

**참여율 계산:**
```
참여율 = (좋아요 수 + 댓글 수) / 조회수 × 100
```

| 참여율 | 보정 배율 | 설명 |
|-------|---------|------|
| 3% 미만 | ×0.9 | 낮은 참여도 |
| 3~5% | ×1.0 | 일반적 참여도 |
| 5~7% | ×1.1 | 높은 참여도 |
| 7% 이상 | ×1.2 | 매우 높은 참여도 |

### 실제 계산 예시

**예시 1: 중형 유튜버**
- 구독자: 20만명
- 평균 조회수: 5만
- 참여율: 4%

계산:
1. 기본 비용 = 50,000 × 10 = 500,000원
2. 구독자 보정 = ×1.0 (10만~50만)
3. 참여율 보정 = ×1.0 (3~5%)
4. **최종 비용 = 500,000원**

**예시 2: 대형 유튜버**
- 구독자: 80만명
- 평균 조회수: 30만
- 참여율: 6%

계산:
1. 기본 비용 = 300,000 × 10 = 3,000,000원
2. 구독자 보정 = ×1.2 (50만~100만)
3. 참여율 보정 = ×1.1 (5~7%)
4. **최종 비용 = 3,960,000원**

### 최종 추천 범위
프로그램은 최종 비용의 ±20% 범위를 추천합니다:
- 최소 비용 = 최종 비용 × 0.8
- 최대 비용 = 최종 비용 × 1.2

## 🔍 표시되는 정보 설명

### 채널 통계
- **구독자 수**: 채널의 총 구독자 수
- **총 동영상 수**: 채널에 업로드된 전체 동영상 개수
- **총 조회수**: 채널의 모든 동영상 누적 조회수

### 최근 영상 분석
- **평균 조회수**: 최근 10개 영상의 평균 조회수
- **평균 참여율**: 최근 10개 영상의 평균 참여율 (%)

### 최근 영상 상세 정보
각 영상별로:
- 제목
- 조회수
- 좋아요 수
- 댓글 수
- 개별 참여율

## 🐛 문제 해결

### Q1: "API 키가 유효하지 않습니다" 오류
**해결 방법:**
1. API 키를 올바르게 복사했는지 확인
2. Google Cloud Console에서 YouTube Data API v3가 활성화되어 있는지 확인
3. API 키가 제한되어 있다면, YouTube Data API v3가 허용되어 있는지 확인

### Q2: "할당량을 초과했습니다" 오류
**원인:** YouTube API는 하루 10,000 단위의 무료 할당량이 있습니다.

**해결 방법:**
1. 내일까지 기다리기 (할당량은 매일 자정(PST)에 초기화)
2. Google Cloud Console에서 결제를 활성화하고 할당량 증가 요청

### Q3: "채널을 찾을 수 없습니다" 오류
**해결 방법:**
1. URL을 다시 확인 (올바른 유튜브 채널 URL인지)
2. 다른 형식의 URL 시도:
   - `@channelname` 형식
   - `channel/ID` 형식

### Q4: 프로그램이 실행되지 않습니다
**해결 방법:**
1. Python이 설치되어 있는지 확인: `python --version`
2. 필요한 패키지가 설치되어 있는지 확인: `pip list`
3. 패키지 재설치: `pip install -r requirements.txt --upgrade`

### Q5: 최근 영상이 표시되지 않습니다
**원인:** 비공개 또는 삭제된 영상은 표시되지 않습니다.

**참고:** 프로그램은 공개된 최근 영상만 분석합니다.

## 📝 주의사항

1. **API 할당량**: 하루 10,000 단위까지 무료로 사용 가능
   - 채널 1개 조회: 약 3-5 단위 사용
   - 하루 약 2,000~3,000개 채널 조회 가능

2. **비용은 참고용**: 산출된 비용은 데이터 기반 추정치입니다.
   - 실제 광고 비용은 다양한 요인에 따라 달라질 수 있습니다
   - 인플루언서와 직접 협의가 필요합니다

3. **개인정보 보호**: API 키는 공개하지 마세요
   - 다른 사람과 공유하지 않기
   - GitHub 등에 업로드하지 않기

4. **정확한 URL 필요**: 채널 URL만 지원합니다
   - 개별 동영상 URL은 지원하지 않음
   - 재생목록 URL은 지원하지 않음
