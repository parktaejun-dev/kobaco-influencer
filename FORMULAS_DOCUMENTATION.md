## 📊 1. 참여율 (Engagement Rate) 계산

**위치:** `youtube_influencer_app.py:206-219`

### 산식
```
참여율(%) = (좋아요 수 + 댓글 수) / 조회수 × 100
```

### 입력 변수
- `likes` (좋아요 수)
- `comments` (댓글 수)
- `views` (조회수)

### 특이사항
- 조회수가 0인 경우 참여율은 0%로 처리
- 소수점 둘째 자리까지 반올림

## 📊 2. 평균 조회수 계산

**위치:** `youtube_influencer_app.py:221-229`

### 산식
```
평균 조회수 = 전체 영상 조회수 합계 / 영상 개수
```

### 입력 변수
- `total_views` (모든 영상의 조회수 합계)
- `video_count` (분석 대상 영상 개수)

### 특이사항
- 정수 나눗셈 사용 (소수점 버림)
- 영상이 없는 경우 0 반환

## 💰 3. CPM 기반 기본 비용 계산

**위치:** `cost_calculator.py:35-38`

### 산식
```
CPM 기본 비용(원) = (평균 조회수 / 1,000) × CPM 단가
```

### 상수
- `CPM 단가` = 39,000원 (1,000뷰당)
  - 2024-2025 글로벌 벤치마크 기준 (약 $30)
  - 이전 97,500원에서 하향 조정

### 예시
- 평균 조회수 100,000회인 경우
- (100,000 / 1,000) × 39,000 = 3,900,000원

## 💰 4. 구독자 티어별 최소 보장 금액

**위치:** `cost_calculator.py:40-51`

### 티어 분류표

| 티어 등급 | 구독자 수 범위 | 최소 보장 금액 (원) |
|---|---|---|
| **Nano** | 1,000 ~ 9,999 | 350,000 |
| **Micro** | 10,000 ~ 99,999 | 2,500,000 |
| **Mid-tier** | 100,000 ~ 499,999 | 5,200,000 |
| **Macro** | 500,000 ~ 999,999 | 19,500,000 |
| **Mega** | 1,000,000+ | 47,500,000 |

### 산식 (조건문)
```python
if 구독자 < 10,000:
    티어 기본액 = 350,000
elif 구독자 < 100,000:
    티어 기본액 = 2,500,000
elif 구독자 < 500,000:
    티어 기본액 = 5,200,000
elif 구독자 < 1,000,000:
    티어 기본액 = 19,500,000
else:  # 1,000,000+
    티어 기본액 = 47,500,000
```

## 💰 5. 기본 비용 선정 (최댓값 선택)

**위치:** `cost_calculator.py:53-54`

### 산식
```
기본 비용 = MAX(CPM 기반 비용, 티어별 최소 보장 금액)
```

### 논리
- CPM 계산값과 티어 최소값 중 **더 높은 금액**을 선택
- 인플루언서가 최소한의 보상을 받도록 보장

### 예시
- CPM 비용: 3,900,000원
- 티어 최소값 (10만 구독자): 2,500,000원
- → 기본 비용 = 3,900,000원

## 📈 6. 참여율 보정 계수

**위치:** `cost_calculator.py:56-78`

### 보정 계수 테이블

| 참여율 범위 | 보정 계수 | 등급 |
|---|---|---|
| **10% 이상** | 1.5 | 최상 |
| **7% ~ 10%** | 1.3 | 매우 높음 |
| **5% ~ 7%** | 1.2 | 높음 |
| **3% ~ 5%** | 1.1 | 양호 |
| **2% ~ 3%** | 1.0 | 보통 |
| **1% ~ 2%** | 0.9 | 낮음 |
| **1% 미만** | 0.85 | 매우 낮음 |

### 산식 (조건문)
```python
if 참여율 >= 10:
    계수 = 1.5
elif 참여율 >= 7:
    계수 = 1.3
elif 참여율 >= 5:
    계수 = 1.2
elif 참여율 >= 3:
    계수 = 1.1
elif 참여율 >= 2:
    계수 = 1.0
elif 참여율 >= 1:
    계수 = 0.9
else:  # < 1%
    계수 = 0.85
```

## 🇰🇷 8. 한국 시장 조정 계수

**위치:** `cost_calculator.py:102-108`

### 산식
```python
if 구독자 < 100,000:
    한국 시장 계수 = 0.85  # 85%
else:
    한국 시장 계수 = 0.75  # 75%
```

### 논리
- **일반 케이스:** 글로벌 대비 75% 수준
- **나노/마이크로 (10만 이하):** 한국 시장에서 더 활발하므로 85% 적용

### 배경
- 한국은 글로벌 대비 광고비 시장 규모가 작음
- 소규모 인플루언서는 로컬 시장에서 영향력이 상대적으로 높음

## 📊 10. 비용 범위 산정 (최소/최대)

**위치:** `youtube_influencer_app.py:361-362`

### 산식
```
최소 비용 = 최종 비용 × 0.85  (85%, -15%)
최대 비용 = 최종 비용 × 1.15  (115%, +15%)
```

### 논리
- 권장 비용 기준 상하 **±15%** 범위 제공
- 협상 여지를 위한 가격 밴드 제시

## 🔄 전체 계산 흐름도

```
1. YouTube API 데이터 수집
   ↓
2. 참여율 계산: (좋아요 + 댓글) / 조회수 × 100
   ↓
3. 평균 조회수 계산: 총 조회수 / 영상 개수
   ↓
4. CPM 기반 비용: (평균 조회수 / 1,000) × 39,000
   ↓
5. 티어별 최소 비용 조회 (구독자 수 기준)
   ↓
6. 기본 비용 = MAX(CPM 비용, 티어 최소 비용)
   ↓
7. 참여율 보정 계수 결정 (0.85 ~ 1.5)
   ↓
8. 글로벌 최종 비용 = 기본 비용 × 참여율 계수
   ↓
9. 한국 시장 조정 계수 결정 (0.75 or 0.85)
   ↓
10. 한국 최종 비용 = 글로벌 비용 × 한국 계수
    ↓
11. 비용 범위 산출: 최종 비용 × (0.85 ~ 1.15)
```
